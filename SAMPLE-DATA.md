## Что произойдет после загрузки:

1. **Автоматически создадутся склады:**
   - Москва (дефолт: 7 дней доставки)
   - Санкт-Петербург (дефолт: 7 дней доставки)
   - Екатеринбург (дефолт: 7 дней доставки)
   - Казань (дефолт: 7 дней доставки)
   - Новосибирск (дефолт: 7 дней доставки)

2. **Рассчитаются все метрики:**
   - Продажи в день для каждого товара
   - Даты запуска сборки
   - Критические товары будут выделены красным
   - Товары требующие внимания - желтым

3. **Будет доступна аналитика:**
   - Дашборд с ближайшими отгрузками
   - Фильтрация по складам
   - Поиск по товарам
   - Статистика по срочным товарам

## Как использовать:

1. Скопируйте данные в Excel или создайте CSV файл
2. Откройте приложение
3. Перейдите на вкладку "Загрузка"
4. Загрузите файл
5. Изучите результаты на всех вкладках

## Примечания:

- **Срочные товары** - товары где "Запуск сборки" уже прошел (показывается как "СРОЧНО!")
- **Товары требующие внимания** - товары с остатком менее чем на 14 дней
- Вы можете изменить дни доставки для каждого склада на вкладке "Склады"
- При изменении дней доставки все расчеты автоматически пересчитаются
